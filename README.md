# Design Canvas

> Web design components

## Usage

Add a single *design-canvas*: it will fill the whole viewport.

Put also some *window-frame*, for example

```html
<design-canvas>
  <window-frame
    position="10,10"
    size="200x400"
    src="path/to/page.html"
  ></window-frame>
</design-canvas>
```

See [demo](https://fibo.github.io/design-canvas).
